# DeepSeek监控助手 v0.1.1

这个版本重点修复 Windows 桌面端“看不到按钮、弹窗无法移动、显示不完整”的问题。

## 下载

- macOS Apple Silicon：`DeepSeek-Monitor-0.1.1-mac-arm64.dmg`
- Windows x64：`DeepSeek-Monitor-0.1.1-win-x64-Setup.exe`

## 本次更新

- Windows 弹窗默认改为更适合桌面托盘的尺寸，并自动限制在屏幕可见区域内。
- 无边框窗口增加顶部拖拽区，按钮和输入框保持可点击。
- 未配置状态下也会显示「配置 Key 开始监控」和「一键体检」，不再让用户误以为功能不可用。
- 修复从主按钮打开设置时，设置浮层被外部点击逻辑立刻关闭的问题。
- 设置浮层保持不透底，避免内容叠在一起显得混乱。
- Windows 安装包固定使用 `Setup.exe` 命名，避免和旧便携包混淆。
- 新增桌面 UI 可用性测试，覆盖窗口尺寸、拖拽区、按钮可见和安装包命名。

## 自检结果

- `npm test`：29 项通过。
- 本地接口冒烟：页面、样式、脚本、状态、设置、余额、体检 API 均正常返回。
- 首次打开模拟：没有 CC Switch 配置时，主按钮显示为「配置 Key 开始监控」。
- Windows 尺寸模拟：420×760 下无横向溢出，关键按钮在首屏可见，内容可滚动。
- 设置浮层模拟：点击主按钮后浮层保持打开，Key 输入框自动聚焦。

## 注意

- Windows 安装包当前未做代码签名，如果 SmartScreen 提醒，请选择信任后运行。
- Windows 上直接克隆 CC Switch 当前 Provider 仍依赖本机 `sqlite3` 命令；没有 sqlite3 时，可以用「保存到 CC Switch」导入链接或「不用 CC Switch，直接使用」。
- 本工具不会保存 prompt 或模型回复，只保存 Token、命中率、状态码、延迟等用量元数据。
